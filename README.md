# 🎬 Movie ETL Pipeline

## 📌 Overview
This project extracts movie data from **AWS S3**, loads it into **Snowflake**, and visualizes insights using **Power BI**.

## 🛠️ Technologies Used
- **AWS S3**: Data storage
- **Snowflake**: Data warehouse
- **Power BI**: Data visualization
- **DAX Queries**: For advanced calculations
